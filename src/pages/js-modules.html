JS Modules

JS Modules are objects that use Javascript and JQuery(Javascript library) to trigger certain functions based on events such has hovering, clicking, scrolling, etc. Carousels, dropdowns, and accordions are great examples of modules.

  • Modals
  • Drawers
  • Accordions
  • Dropdowns
  • Tooltips
  • Carousels

Modals

Modals are custom pop-up windows that are triggered by either a button or link mouse click event. They are commonly used to contain videos, images, forms, maps, and extra content. When a modal is active, an overlay covers the whole page (except for the modal content container) preventing the user from interacting with the non-modal content until the modal is closed. Two common ways of closing a modal is to have a close button and/or create a close function on the modal overlay. The best practice is to use both closing function as to give more options to the user.

JS

/* Modal Module ***************************************************************************/
$(document).ready(function() {
    var modalArc = '.js-modalArc'; //Currently placed on body tag. If more than one modal, is on a page, 
                                //you will need to place this on a parent div that contains
                                //both the trigger and modal while making sure that modal parents 
                                //are not children of other modal parents. 
    var trigger = '.js-modalTrigger';
    var modal = '.js-modal';
    var content = '.js-modal-content';
    var close = '.js-modal-close';
    var maskClose = '.js-modal-mask_close';
    var duration = 300;

    $(modalArc).find(trigger).click(function() {
        $(this).parents(modalArc).find(modal).fadeIn(duration);
    });
    $(modal).find(close).click(function() {
        $(this).parent(content).parent(modal).fadeOut(duration);
    });
    $(maskClose).click(function() {
        $(this).parent(modal).fadeOut(duration);
    });
});

Drawers

Drawers show and hide content using a trigger based on an mouse event. The content is displayed directly below the trigger, and a slide up/down animation is generally applied to the function. Drawers are independently triggered and do not effect the display of other drawers (opening one drawer will not close any other drawers on the page). Drawers can also be stacked together to create an accordion like layout but keep their functionality independent. Drawers are a common module used to show/hide examples for this projects.

Accordions

Accordions look and behave similar to a set of drawers stacked together. Their sections, usually called accordion items show and hide content using a trigger based on an mouse event. The content is displayed directly below the trigger, and a slide up/down animation is generally applied to the function. Unlike drawers, accordion items are dependent to their parent accordion and do effect the display of other sibling accordion items (opening one accordion item will close all other sibling accordion items).

JS

/* Accordion Module ***************************************************************************/
$(document).ready(Accordion = function() {
    var accordion = '.js-accordion';
    var item = '.js-accordion-item';
    var trigger = '.js-accordion-trigger';
    var content = '.js-accordion-content';
    var contentActive = '.js-accordion-content.isActive';
    var icon = 'i';
    var duration = 300;

    // Styles accordion item if you choose to start with an open accordion on load //
    $(accordion).find('.isActive').parent(item).find('i').addClass('isActive');
    $(accordion).find('.isActive').css("display", "block");

    // Click function logic //
    $(accordion).find(item).find(trigger).click(function(event) {

        $(accordion).find(contentActive).slideUp(duration);
        $(accordion).find(icon).removeClass('isActive');

        if ($(this).next(content).hasClass('isActive')) {
            $(content).removeClass('isActive');
            $(this).find(icon).removeClass('isActive');
        } else {
            $(content).removeClass('isActive');
            $(this).next(content).slideDown(duration);
            $(this).next(content).addClass('isActive');
            $(this).find(icon).addClass('isActive');
        }

    });
});

Dropdowns

A dropdown behaves similarly to a drawer, but instead of pushing all other content below its own, the dropdown content is displayed on top of other content (positioned absolutely). Dropdowns are most commonly used for mobile navigation and desktop nested navigation. A mouse click (necessary for mobile and tablet) or hover (optional for desktop) event on a trigger is used to open the dropdown content. If a hover trigger is used for desktop, there will be a possibility of conflict between the click and hover function on a tablet device when flipping between landscape and portrait view. This conflict is caused by the media query width landing in between a device's portrait and landscape screen width (if you are on a desktop and want to see this conflict in action, shrink your viewing window below 768px and then try clicking on the example dropdowns below, notice that they do not open. Expand back to full screen and the hover state will work again).

Carousels

Carousels (also known as image sliders) are image slideshow modules. They comprise a list of media items(images or video) that show one media item at a time. There are on click triggers that navigate to each media item. The most common triggers are the left/right nav arrows that cycle to the previous/next media item respectively. Another common trigger set are the bullets/thumbnails that are usually located below the active media item, and these on click triggers cycle you directly to its respective media item. Carousels are very complicated modules and take a lot of development time to create from scratch. A plugin carousel like FlexSlider can save precious time and add a more robust carousel to fit multiple needs. FlexSlider is the carousel plugin of choice for the latest Wiley websites.

Setup Steps

The Javascript is already written with the FlexSlider plugin. There are only a few steps needed to set the plugin up:

  1. Download the FlexSlider plugin, unzip the contents.
  2. Place the slider folder into the statics root directory. This folder contains the css, js, and icons for the slider.

  3. Setup your css slider.css path in your html head with your other css paths. Make sure the href path is correct.

    <!-- STYLESHEETS -->
<link rel="stylesheet" href="css/css/modern.css">
<link rel="stylesheet" href="slider/flexslider.css" type="text/css">
<!--[if lt IE 9]>
    <link rel="stylesheet" href="css/CSS/legacy.css">
<![endif]-->

  4. Setup your js slider path below the footer with your other js paths. Make sure the jquery.flexslider.js path is below the jquery.js path, or the slider will not work due to its reliance on jquery. Make sure the href path is correct.

    <!-- SCRIPTS -->
<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.3/jquery.min.js"></script>
<script src="js/plugins.js"></script>
<script src="js/main.js"></script>
<script src="slider/jquery.flexslider.js"></script>

</body>

  5. Set up the slider load function. Place this function directly below the jquery.flexslider.js path. If you place this function above it, the slider will not work. Notice that there are many variables in the load function. These variables allow you to customize the functionality of the slider. These are a few examples, but there are many more variable functions that can be used.

    <!-- SCRIPTS -->
<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.3/jquery.min.js"></script>
<script src="js/plugins.js"></script>
<script src="js/main.js"></script>
<script src="slider/jquery.flexslider.js"></script>

      $('.flexslider').flexslider({
          animationSpeed: 800,
          touch: true,
          pauseOnHover: true,
          pauseOnAction: true,
          startAt:0,
          animation: "fade"
        });
    });
</script>
